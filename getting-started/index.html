    Getting Started¶

    Install Python Requirements¶

    This project uses pipenv. Install dependencies with:

    pipenv install

    Activate the virtual environment:

    pipenv shell

    Or run a single command in the virtual environment, e.g.:

    pipenv run python manage.py runserver

    Create and Migrate Database¶

    Create a new database for your project:

    pipenv shell
python manage.py migrate

    This will create the necessary tables in your database.

    Create Admin Account¶

    Create a new admin account:

    pipenv run python manage.py createsuperuser

    Follow the prompts to set up your admin account.

    Dump Data to Fixture¶

    python manage.py dumpdata --exclude=admin.logentry --exclude=contenttypes > .\api\fixtures\fixtures.json

    Load Data from Fixtures¶

    Load sample data from fixtures:

    pipenv run python manage.py loaddata fixtures/*.json

    Run the Server¶

    Run the development server:

    python manage.py runserver

    Run Huey consumer for async/periodic tasks¶

    python manage.py run_huey

    Access Swagger UI¶

    Access the Swagger UI to explore the API: http://localhost:8000/docs/swagger-ui/

    Access Django Admin UI¶

    Access the Django admin UI to manage your data: http://localhost:8000/admin/
